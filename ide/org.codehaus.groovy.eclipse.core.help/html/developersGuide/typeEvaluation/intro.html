Type Evaluation

GroovyEclipse contains a type evaluator class that can be customized for different purposes. It is currently used for code completion in the source editor and in the type inference engine. Customizing the state of the type evaluator allows for code completion in other areas:
  • A Groovy Console embedded into Eclipse, with code completion for variables in the binding.
  • A Groovy Script evaluator, with completion for variables at a debugger break point.
The type evaluator can be found at org.codehaus.eclipse.core.types.TypeEvaluator.
