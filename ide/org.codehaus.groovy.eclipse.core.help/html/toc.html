The GroovyEclipse Developers Guide

This guide descibes the various extension points which can be used to extend GroovyEclipse.
The extension points can be implemented to enhance GroovyEclipse:
  • To create new features without disturbing the source in the existing plugins. These can be rolled into the main source tree at a later stage.
  • To add experimental features.
  • To customize GroovyEclipse to a specific domain, say Grails development.
  • To customize GroovyEclipse for a specific project.
